# PROJECT_CANON.md - 项目宪章

**版本**: 0.1.0
**创建日期**: 2026-02-03
**状态**: 生效

## 1. 项目概述

**项目名称**：青少年 iPad AI 英语学习系统
**目标用户**：13-18 岁学生
**核心目标**：AI 出题 + 学习分析 + 家长/老师监管 + AI 自动迭代升级

## 2. 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **客户端** | Swift + SwiftUI | 原生 iPadOS 体验 |
| **服务端** | Node.js + Express | 轻量 API 网关 |
| **AI 引擎** | MiniMax (主) + Gemini 3 (辅) | 混合出题与分析 |
| **数据库** | SQLite (MVP) / Postgres (生产) | 数据持久化 |
| **存储** | localStorage (Client) | 轻量数据 |

## 3. 核心角色

- **学生 (Student)**：学习主体，接收 AI 出题与讲解。
- **家长 (Parent)**：查看学习报告与专注度统计。
- **老师 (Teacher)**：管理学生，配置学习任务。

## 4. 核心功能 (MVP 0.1.0)

1.  **水平测试**：5-10 分钟自适应测试。
2.  **AI 学习路径**：每日 15-25 分钟动态学习计划。
3.  **动态出题**：基于错因分析的针对性练习。
4.  **能力画像**：五维能力雷达图（词汇/语法/阅读/听力/口语）。
5.  **专注度检测**：基于摄像头的在场状态统计。

## 5. 约束与铁律

- **隐私优先**：摄像头数据仅本地处理，不上传不存储。
- **成本控制**：AI 调用必须走代理网关，记录 Token 消耗。
- **迭代模式**：A 回合（规划） -> B 回合（交付），严禁盲目编码。
