## 项目状态
- **阶段**：B 回合交付中 (0.1.0 MVP)
- **最后更新**：2026-02-03

## 技术栈
- **客户端**：Swift + SwiftUI (iPadOS)
- **服务端**：Node.js + Express
- **AI**：MiniMax / Gemini 3

## 快速开始

### 服务端
```bash
cd server
npm install
npm run dev
```

### 客户端
打开 `client/ios/aixuexi2026.xcodeproj` (待生成)。
